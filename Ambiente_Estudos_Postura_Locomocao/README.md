# Ambiente de Estudos - Postura e Locomocao

Este pacote foi criado para estudar a materia do PDF `Postura e Locomocao.pdf`.

## Arquivo principal

Abra:

`index.html`

O ambiente roda direto no navegador e nao precisa instalar nada.

## Central HTML

Abra tambem:

`MATERIAIS.html`

Essa pagina reune todos os materiais em HTML para facilitar o compartilhamento. Ela foi corrigida para abrir os materiais por conteudo embutido, sem depender de links relativos frageis.

Na raiz do pacote tambem existe:

`INICIAR_AQUI.html`

Use esse arquivo como primeira opcao ao compartilhar o ZIP.

## O que tem dentro

- Modulos guiados por assunto.
- Leitor das laminas originais do PDF.
- Aba interna de anatomia 3D 360 graus inspirada na lamina de areas corticais.
- Experiencia 3D das laminas com perspectiva e modo automatico.
- Flashcards interativos com progresso salvo no navegador.
- Quiz de fixacao.
- Casos rapidos de aplicacao clinica.
- Tabela comparativa NMS x NMI x Cerebelar.
- Simulador conceitual de integracao sensorial.

## Materiais extras

- `GUIA_DE_ESTUDOS.md`: explicacao organizada da aula.
- `GUIA_DE_ESTUDOS.html`: versao HTML compartilhavel do guia.
- `QUESTOES_REVISAO.md`: perguntas, casos e respostas-modelo.
- `QUESTOES_REVISAO.html`: versao HTML compartilhavel das questoes.
- `FLASHCARDS.tsv`: flashcards exportaveis para Anki/Quizlet.
- `FLASHCARDS.html`: versao HTML interativa dos flashcards.
- `ANATOMIA_3D.html`: arquivo de apoio do modelo 3D; o acesso principal fica dentro do ambiente completo.
- `LAMINAS_3D.html`: visualizacao 3D compartilhavel das 21 laminas.
- `README.html`: instrucoes em HTML.

## Como compartilhar

Para compartilhar apenas teoria, questoes ou flashcards, envie o HTML correspondente.

Para compartilhar o ambiente completo com as laminas, envie a pasta `Ambiente_Estudos_Postura_Locomocao` junto com a pasta `page_images`, mantendo as duas na mesma pasta-mae. Se houver um arquivo `.zip` gerado, ele ja contem a estrutura correta.

## Melhor forma de usar

1. Estude os modulos no `index.html`.
2. Reveja as laminas relacionadas em cada modulo.
3. Faca os flashcards ate responder sem olhar.
4. Resolva o quiz.
5. Termine pelos casos clinicos.
6. Leia o `GUIA_DE_ESTUDOS.md` antes da prova.

Instruções HTML compartilháveis

Ambiente de Estudos

Como usar e compartilhar o pacote de Postura e Locomoção.

Ambiente principal Materiais Guia Questões Flashcards Lâminas 3D

Arquivo principal: abra index.html no navegador. O ambiente roda localmente e não precisa instalar nada.

O que tem dentro

  • Módulos guiados por assunto.
  • Leitor das lâminas originais do PDF.
  • Aba interna de anatomia 3D em 360 graus inspirada na lâmina de áreas corticais.
  • Experiência 3D das lâminas com perspectiva, rotação e modo automático.
  • Flashcards interativos com progresso salvo no navegador.
  • Quiz de fixação.
  • Casos rápidos de aplicação clínica.
  • Tabela comparativa NMS x NMI x Cerebelar.
  • Simulador conceitual de integração sensorial.

Materiais HTML

  • GUIA_DE_ESTUDOS.html: explicação organizada da aula.
  • QUESTOES_REVISAO.html: perguntas, casos e respostas-modelo.
  • FLASHCARDS.html: flashcards interativos em página única.
  • ANATOMIA_3D.html: arquivo de apoio do modelo 3D; o acesso principal fica dentro do ambiente completo.
  • LAMINAS_3D.html: visualização 3D compartilhável das 21 lâminas.
  • MATERIAIS.html: central de links para todos os arquivos.

Como compartilhar

Para compartilhar apenas teoria, questões ou flashcards, envie o HTML correspondente. Esses arquivos funcionam sozinhos.

Para compartilhar o ambiente completo com as lâminas originais, envie a pasta Ambiente_Estudos_Postura_Locomocao junto com a pasta page_images, mantendo as duas na mesma pasta-mãe.

Melhor forma de usar

  1. Estude os módulos no index.html.
  2. Reveja as lâminas relacionadas em cada módulo.
  3. Faça os flashcards até responder sem olhar.
  4. Resolva o quiz.
  5. Termine pelos casos clínicos.
  6. Leia o guia antes da prova.
